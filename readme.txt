A WordPres WooCommerce plugin to add the checkbox required to accept Terms of Use and Privacy Policy on WooCommerce register form.

== Description ==

Add the checkbox with the following text to the registration form:

    I have read and consent to the Terms of Use and Privacy policy *

This checkbox is required to accept when user register new account.

== Installation ==

Install the plugin as usual.

Activate the plugin.

Create, set and pulish the "Terms of Use" and "Privacy Policy" pages.

Set "Terms of Use" page (for Dokan).

Set "Privacy Policy" page in the admin Settings - Privacy settings page.

In the admin dashboard swith to the WooCommerce - Settings - Accounts & Privacy and remove text from the "Registration privacy policy" and "Checkout privacy policy" fields to remove it form the registration form.
